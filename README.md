Реализованные механики:
1.  GameInstance – используется для хранения состояния игры, хранит в себе текущий номер комнаты (уровень подземелья), количество полученной валюты, содержит список комнат для процедурной генерации.
2.  Система порталов – реализована логика перехода между уровнями. В начале прохождения выбирается количество этажей подземелья (от 6 до 10). При входе в портал, система проверяет текущий этаж подземелья. Если этаж последний – то героя переносит на уровень с боссом и его приспешниками. Иначе – игрока переносит на случайный уровень из массива всех созданных.
3.  Реализованы 3 типа врагов:
a.  Дроны-снайперы дальнего боя – стреляют в игрока, имеют малое количество здоровья.
b.  Дроны ближнего боя – летают по комнате в поисках игрока. Имеют три состояния – патрулирование с использованием умного зрения, преследования и смерти. После смерти у дронов включается физика и они падают на землю, а после исчезают. AI-поиска игрока реализован на базе навигационной сетки NavMesh и компонента PawnSensing. При столковении с игроком, последнему наносится урон. После получения урона игрок отбрасывается.
c.  Дрон-Босс – большой дрон дальнего боя, имеет повышенное количество здоровья.
d.  На каждом уровне используется Select для случайного выбора типа врага в комнате.
4.  Реализована ловушка – при входе на первый этап подземелья дверь за игроком закрывается.
5.  Реализовано два HUD’а – WBP_HUD используется для отображения здоровья через ProgressBar, количества патронов и количества денег. Деньги начисляются за убийство врагов. WBP_WinScreen показывается после убийства всех врагов в комнате босса и позволяет выйти из игры.
6.  Здоровье и патроны восстанавливаются при переходе на новый уровень. Решение не запихивать их в GameInstance было принято сознательно, тк автомат для траты денег (восстановления патронов и здоровья) не был реализован, игру было физически очень сложно пройти.
7.  VFX – использована система Niagara для создания эффекта пыли внутри уровней подземелья.
8.  Добавлены текстуры для стен подземелий и дверей, добавлен красивый визуальный эффект для порталов.
9.  Сложность каждого этажа кроме последнего (на последнем всегда будет сложный босс) выбирается случайно и достигается путем рандомизации типов врагов, форм комнат и количества врагов.
